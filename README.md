# 使用说明
下载项目后，进入项目目录
打开命令行：
1. npm install安装依赖
2. 修改./config/config.js里的相关配置信息，你需要安装mysql，新建数据库，并配置好用户权限
3. node serve.js启动服务

启动后，根据命令行给出的地址，在本地浏览器里就可以访问啦
如果需要服务器启动，你需要绑定域名并配置反向代理哦
具体可以参考网络上的文章

有不懂的问题可以前往文末的社区交流哈
